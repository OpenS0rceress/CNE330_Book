Write a function, compare_ip_length, that takes two IP addresses as strings, and returns the longer IP address. If they are the same length, return either IP address.

As an example, if you passed in the IPs 192.168.1.1 and 192.168.1.101, the function would return 192.168.1.101.

The len function, when applied to a string, returns the number of characters in a string.
