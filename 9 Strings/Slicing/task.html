Write two functions, get_first_octet and get_third_octet.

get_first_octet takes an IP address in, and returns the number first octet of the IP address

As an example, if you passed in the IP 192.168.76.1 the function would return 192

get_third_octet takes an IP address in, and returns the number third octet of the IP address

As an example, if you passed in the IP 192.168.76.1 the function would return 76

The output of the functions should only have numbers, no periods.

The slice operator [n:m] returns the part of the string from the n’th character to the m’th character, including the first but excluding the last. In other words, start with the character at index n and go up to but do not include the character at index m. This behavior may seem counter-intuitive but if you recall the range function, it did not include its end point either.

If you omit the first index (before the colon), the slice starts at the beginning of the string. If you omit the second index, the slice goes to the end of the string.
