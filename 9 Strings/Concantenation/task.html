Write a function, create_ip_address, that takes in four IP octets as integers and returns the full IP address. If they are the same length, return either IP address.

As an example, if you passed in the numbers 192, 168, 76, and 1 the function would return 192.168.76.1.

Make sure to convert the variables to strings

Interestingly, the + operator does work with strings, but for strings, the + operator represents concatenation, not addition. Concatenation means joining the two operands by linking them end-to-end.
